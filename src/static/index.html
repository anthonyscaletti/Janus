Janus

Janus is a Machine Learning API used to conduct time series predictions and forecasting on supplied data. It utilizes a Long Short-Term Memory Recurring Neural Network model to make these predictions.

API Endpoints:

  • GET: "/" : Entry page with Definitions

  • GET: "/entry" : Entry page with Definitions

  • POST: "/api/janus" : API Endpoint to execute time series forecasting. Supply data via the body of this request in the format defined below.

Request Body Data Format:

  • X Axis "x" : An Array containing timestamps in the format MM/DD/YYYY

  • Y Axis: "y" : An Array containing historical data mapping to timestamps in the x axis

  • Predicted Days Count: "dayCount" : Number of days in the future that you wish to predict

Sample Request Body:

{

"x": ["04/01/2020", "04/02/2020", "04/03/2020", "04/04/2020", "04/05/2020"],

"y": [1, 2, 3, 4, 5],

"dayCount": 5

}

Sample Response:

{

"x": ["04/06/2020", "04/07/2020", "04/08/2020", "04/09/2020", "04/10/2020"],

"y": [6, 7, 8, 9, 10]

"dayCount": 5

}
